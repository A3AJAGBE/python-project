# DrugInfoSys

DrugInfoSys is Developed and Designed using Flask (Python), MySql, HTML, CSS, JavaScript and jQuery.

## About
This project is a search engine for users to get real information about drugs (medication) straight from the
pharmaceutical company. The data will come from the company that made the medicine,
while an administrator will monitor the data and system.
