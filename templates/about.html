DrugInfoSys

DrugInfoSys is Developed and Designed to complete Programming for Information Systems class project, done using Flask (Python), MySql, HTML, CSS, JavaScript and jQuery.

This project is a search engine for users to get real information about drugs (medication) straight from the pharmaceutical company. The data will come from the company that made the medicine, while an administrator will monitor the data and system.

This system and the report attached is completed by Aminat Ajagbe (10541380), Saikrishna Sattayya Dumpala (10540452), and Venkata Hanuma Sai Krishna Pothineni (10539362) students of Dublin Business School, studying MSc. Information Systems with computing.
